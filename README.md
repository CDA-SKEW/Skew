## Installation Front-end

```bash
  # React
  npx create-react-app my-app
  cd my-app
  +
  # Redux
  npm install redux

  # Yarn
  yarn add redux
```
## Installation Back-end

```bash
  # Node.js
  sudo apt-get install nodejs npm

  # Express.js
  npm install express

  # Core
  npm i core-js

  # SGBD Mysql
  sudo apt-get install mysql-shell
```
